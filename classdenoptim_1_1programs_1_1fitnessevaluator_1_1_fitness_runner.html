Detailed Description

Stand-alone fitness provider.

This class implements a method that allows to run a fitness evaluation in a stand-alone fashion. The configuration of the fitness provider is given as a DENOPTIM's input parameter file, that is in all equal to the parameters file used by GARunner and FragSpaceExplorer.
